# benmorrell.co

Ben Morrell's personal portfolio site. Next.js + Tailwind, deployed on Vercel.

Implements the brand **Visual Identity** (petrol-teal `#0F6E56`, Fraunces + Inter, ink on paper)
defined canonically in the Obsidian vault (`Job Finder/Profile/Visual Identity.md`). Content is
grounded in that vault (Brand, Building & AI, STAR Stories, Master CV) but authored statically here
— **no CMS, and the vault is never copied into this repo.**

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build

```bash
npm run build && npm start
```

## Deploy

Connected to Vercel (auto-deploys `main`). Point the `benmorrell.co` domain at the Vercel project.
